## 系统概述

### 核心功能

构建一个多Agent协作的编码聊天室系统，多个AI Agent在共享工作空间中协作完成编码任务。**Agent数量可配置，默认配置3个Agent**：

- **Claude**（架构设计编码）- OpenRouter API
- **Codex**（代码审查）- OpenRouter API
- **Gemini**（创意发散和视觉设计）- OpenRouter API

### 系统特点

- ✅ 实时群聊式交互（类似Slack/Discord）
- ✅ Agent数量可配置（支持动态添加/移除）
- ✅ @提及功能（@Claude、@Codex、@Gemini）
- ✅ Agent主动感知消息并决策是否响应
- ✅ 共享工作空间和记忆（无需重复上下文）
- ✅ 智能记忆管理（短期/长期/共享）
- ✅ 上下文持久化
- ✅ 代码生成与存储

### 聊天室交互模式

**群聊模式**（类似多人协作）：

```
用户: @Claude 帮我设计一个用户登录系统
Claude: 好的，我来设计架构...（生成代码）
Gemini: 我觉得登录界面可以这样设计...（主动参与）
用户: @Codex 审查一下Claude的代码
Codex: 代码整体不错，建议改进...
```

**特性**：

1. 用户用 `@模型名` 直接召唤
2. Agent可以主动响应（感知到与自己相关的消息）
3. 所有成员共享同一个上下文

### 用户角色

| 角色         | 描述               | 权限                             |
| ------------ | ------------------ | -------------------------------- |
| **普通用户** | 提出需求、管理会话 | 创建会话、@召唤Agent、查看结果   |
| **管理员**   | 系统管理           | 全部权限 + Agent配置管理         |
| **AI Agent** | 自主决策并响应     | 感知消息、主动响应、访问共享记忆 |

### 1.2 功能需求

#### FR-1.1 Agent注册与发现

**需求描述**：系统启动时根据配置自动注册Agent，支持动态添加/移除和健康检查。

**默认Agent配置**：

- **Claude**（角色：架构设计、编码实现）- OpenRouter API
- **Codex**（角色：代码审查、质量把控）- OpenRouter API
- **Gemini**（角色：创意发散、视觉设计）- OpenRouter API

**验收标准**：

- ✅ 从配置文件加载Agent列表（支持任意数量）
- ✅ 每30秒执行一次健康检查
- ✅ Agent状态实时更新（在线/忙碌/离线）
- ✅ 支持热插拔（运行时添加/移除Agent）
- ✅ 每个Agent有独立的角色定位和能力描述

#### FR-1.2 统一调用接口

**需求描述**：提供统一的Agent调用接口，屏蔽底层差异（CLI vs HTTP API）。

**接口定义**：

```typescript
interface ILLMAdapter {
  /**
   * Agent基本信息
   */
  readonly id: string; // Agent唯一标识（如: claude-001）
  readonly name: string; // Agent显示名称（如: Claude）
  readonly model: string; // 模型名称（如: claude-3-sonnet）
  readonly type: 'claude' | 'codex' | 'gemini' | 'custom'; // Agent类型
  readonly role: string; // Agent角色定位（如: "架构设计与编码"）
  readonly capabilities: string[]; // 能力列表（如: ["代码生成", "架构设计"]）
  readonly callType: 'cli' | 'http'; // 调用方式

  /**
   * 生成响应（非流式）
   * @param prompt 用户提示词
   * @param context 上下文信息
   * @returns 生成的响应
   */
  generate(prompt: string, context: AgentContext): Promise<AgentResponse>;

  /**
   * 流式生成响应
   * @param prompt 用户提示词
   * @param context 上下文信息
   * @returns 响应流
   */
  streamGenerate(prompt: string, context: AgentContext): AsyncGenerator<string, void, unknown>;

  /**
   * 判断是否应该响应消息（主动决策）
   * @param message 聊天室消息
   * @param context 上下文信息
   * @returns 是否应该响应 + 响应理由
   */
  shouldRespond(message: ChatMessage, context: AgentContext): Promise<{ should: boolean; reason?: string }>;

  /**
   * 健康检查
   * @returns 是否健康
   */
  healthCheck(): Promise<boolean>;

  /**
   * 获取Agent当前状态
   */
  getStatus(): AgentStatus;
}

// 数据结构
interface AgentContext {
  sessionId: string; // 会话ID
  userId: string; // 用户ID
  conversationHistory: Message[]; // 对话历史
  sharedMemory?: Record<string, any>; // 共享记忆
  metadata?: Record<string, any>; // 元数据
}

interface AgentResponse {
  content: string; // 生成内容
  tokenUsage: {
    prompt: number;
    completion: number;
    total: number;
  };
  metadata?: Record<string, any>; // 额外信息
  timestamp: Date;
}

enum AgentStatus {
  IDLE = 'idle', // 空闲
  BUSY = 'busy', // 忙碌
  ERROR = 'error', // 错误
  OFFLINE = 'offline', // 离线
}
```

#### FR-1.3 Claude Agent实现（HTTP API）

**需求描述**：通过OpenRouter API调用Claude模型，负责架构设计和编码实现。

**技术实现**：

```typescript
@Injectable()
export class ClaudeAdapter implements ILLMAdapter {
  readonly id = 'claude-001';
  readonly name = 'Claude';
  readonly model = 'anthropic/claude-3-sonnet';
  readonly type = 'claude';
  readonly role = '架构设计与编码实现';
  readonly capabilities = ['架构设计', '代码生成', '技术选型', '重构'];
  readonly callType = 'http';

  constructor(
    private readonly httpClient: HttpService,
    private readonly config: ConfigService,
  ) {}

  async generate(prompt: string, context: AgentContext): Promise<AgentResponse> {
    const apiUrl = 'https://openrouter.ai/api/v1/chat/completions';
    const apiKey = this.config.get('OPENROUTER_API_KEY');

    // 构建请求体
    const requestBody = {
      model: this.model,
      messages: [
        {
          role: 'system',
          content: `你是Claude，一个专业的软件架构师和编码专家。
你的职责是：
1. 设计系统架构
2. 编写高质量代码
3. 提供技术选型建议
4. 进行代码重构

当前上下文：${JSON.stringify(context.sharedMemory)}`,
        },
        ...this.buildContextMessages(context),
        { role: 'user', content: prompt },
      ],
      temperature: 0.7,
      max_tokens: 4000,
    };

    // 发送HTTP请求
    const response = await this.httpClient
      .post(apiUrl, requestBody, {
        headers: {
          Authorization: `Bearer ${apiKey}`,
          'Content-Type': 'application/json',
          'HTTP-Referer': 'https://lobster.com',
          'X-Title': 'Lobster Coding Assistant',
        },
      })
      .toPromise();

    return this.parseResponse(response.data);
  }

  /**
   * 判断是否应该响应（Claude主动决策）
   */
  async shouldRespond(message: ChatMessage, context: AgentContext): Promise<{ should: boolean; reason?: string }> {
    // Claude响应条件：
    // 1. 被@提及
    if (message.content.includes('@Claude')) {
      return { should: true, reason: '被直接@提及' };
    }

    // 2. 消息涉及架构、设计、编码关键词
    const keywords = ['架构', '设计', '实现', '代码', '开发', '重构', '优化'];
    if (keywords.some((k) => message.content.includes(k))) {
      return { should: true, reason: '消息涉及我的专业领域（架构设计）' };
    }

    // 3. 其他Agent提到我
    if (message.content.includes('Claude')) {
      return { should: true, reason: '其他成员提到我' };
    }

    return { should: false };
  }
}
```

**动态Agent管理服务**：

```typescript
@Injectable()
export class AgentConfigService {
  private agents: Map<string, ILLMAdapter> = new Map();

  /**
   * 从配置文件加载Agent
   */
  async loadAgentsFromConfig(): Promise<void> {
    const config = await this.loadConfig('agents.config.json');

    for (const agentConfig of config.agents) {
      if (!agentConfig.enabled) continue;

      // 根据类型创建对应的Adapter
      const adapter = this.createAdapter(agentConfig);
      this.agents.set(agentConfig.id, adapter);
    }

    console.log(`已加载 ${this.agents.size} 个Agent`);
  }

  /**
   * 动态添加Agent
   */
  async addAgent(config: AgentConfig): Promise<void> {
    const adapter = this.createAdapter(config);
    this.agents.set(config.id, adapter);

    // 保存到配置文件
    await this.saveAgentConfig(config);

    console.log(`已添加Agent: ${config.name}`);
  }

  /**
   * 移除Agent
   */
  async removeAgent(agentId: string): Promise<void> {
    this.agents.delete(agentId);
    await this.removeAgentConfig(agentId);

    console.log(`已移除Agent: ${agentId}`);
  }

  /**
   * 获取所有Agent
   */
  getAllAgents(): ILLMAdapter[] {
    return Array.from(this.agents.values());
  }

  /**
   * 根据ID获取Agent
   */
  getAgent(agentId: string): ILLMAdapter {
    const agent = this.agents.get(agentId);
    if (!agent) {
      throw new NotFoundException(`Agent ${agentId} not found`);
    }
    return agent;
  }

  /**
   * 根据名称获取Agent（支持@提及）
   */
  getAgentByName(name: string): ILLMAdapter {
    const agent = Array.from(this.agents.values()).find((a) => a.name.toLowerCase() === name.toLowerCase());
    if (!agent) {
      throw new NotFoundException(`Agent ${name} not found`);
    }
    return agent;
  }

  private createAdapter(config: AgentConfig): ILLMAdapter {
    switch (config.type) {
      case 'claude':
        return new ClaudeAdapter(config);
      case 'codex':
        return new CodexAdapter(config);
      case 'gemini':
        return new GeminiAdapter(config);
      default:
        throw new Error(`Unknown agent type: ${config.type}`);
    }
  }
}
```

#### FR-2.3 消息处理与Agent主动响应

**需求描述**：处理用户消息，解析@提及，让所有Agent主动决策是否响应。

**核心逻辑**：

1. 用户发送消息到聊天室
2. 系统广播消息给所有成员
3. **所有在线Agent同时收到消息通知**
4. **每个Agent独立决策是否要响应**（调用 `shouldRespond()` 方法）
5. 决定响应的Agent开始生成内容并流式推送

**实现**：

```typescript
@WebSocketGateway({ namespace: '/chat' })
export class ChatGateway {
  constructor(
    private readonly chatService: ChatService,
    private readonly agentConfigService: AgentConfigService,
    private readonly memoryService: MemoryService,
  ) {}

  @SubscribeMessage('message:send')
  async handleMessage(@ConnectedSocket() client: Socket, @MessageBody() data: { content: string; sessionId: string }) {
    const userId = this.extractUserId(client);

    // 1. 解析@提及
    const mentionedAgents = this.parseMentions(data.content);

    // 2. 保存用户消息
    const message = await this.chatService.saveMessage({
      sessionId: data.sessionId,
      userId,
      role: 'user',
      content: data.content,
      metadata: { mentionedAgents },
    });

    // 3. 广播给房间所有成员（包括前端用户）
    this.server.to(`session:${data.sessionId}`).emit('message:received', message);

    // 4. 触发所有Agent的决策流程（异步并行）
    this.triggerAgentDecisionMaking(message, data.sessionId);
  }

  /**
   * 解析@提及
   */
  private parseMentions(content: string): string[] {
    const mentionPattern = /@(\w+)/g;
    const matches = content.matchAll(mentionPattern);
    return Array.from(matches, (m) => m[1]);
  }

  /**
   * 触发所有Agent的决策流程
   */
  private async triggerAgentDecisionMaking(message: Message, sessionId: string): Promise<void> {
    // 获取所有在线Agent
    const agents = this.agentConfigService.getAllAgents();

    // 获取共享上下文
    const context = await this.memoryService.getSessionContext(sessionId);

    // 并行询问所有Agent是否要响应
    const decisionPromises = agents.map(async (agent) => {
      try {
        const decision = await agent.shouldRespond(message, context);

        if (decision.should) {
          // 通知前端：Agent决定响应
          this.server.to(`session:${sessionId}`).emit('agent:thinking', {
            agentId: agent.id,
            agentName: agent.name,
            reason: decision.reason,
            timestamp: new Date(),
          });

          // 生成响应（异步）
          this.generateAgentResponse(agent, message, sessionId, context);
        } else {
          // 通知前端：Agent决定不响应（可选，避免UI过于嘈杂）
          // this.server.to(`session:${sessionId}`).emit('agent:skip', {
          //   agentId: agent.id,
          //   agentName: agent.name,
          //   reason: decision.reason || '与我无关',
          //   timestamp: new Date()
          // });
        }
      } catch (error) {
        console.error(`Agent ${agent.name} 决策失败:`, error);
      }
    });

    // 等待所有决策完成（不阻塞消息处理）
    Promise.all(decisionPromises);
  }

  /**
   * 生成Agent响应
   */
  private async generateAgentResponse(
    agent: ILLMAdapter,
    message: Message,
    sessionId: string,
    context: AgentContext,
  ): Promise<void> {
    try {
      // 通知开始
      this.server.to(`session:${sessionId}`).emit('agent:start', {
        agentId: agent.id,
        agentName: agent.name,
        timestamp: new Date(),
      });

      // 构建提示词（包含角色定位）
      const prompt = this.buildPromptWithRole(agent, message.content, context);

      // 流式生成
      const stream = agent.streamGenerate(prompt, context);

      let fullContent = '';
      for await (const delta of stream) {
        fullContent += delta;

        // 实时推送增量
        this.server.to(`session:${sessionId}`).emit('agent:stream', {
          agentId: agent.id,
          agentName: agent.name,
          delta,
          timestamp: new Date(),
        });
      }

      // 保存Agent响应到数据库和记忆
      await this.chatService.saveMessage({
        sessionId,
        agentId: agent.id,
        role: 'assistant',
        content: fullContent,
      });

      await this.memoryService.appendToShortTerm(sessionId, {
        role: 'assistant',
        content: fullContent,
        agentId: agent.id,
      });

      // 通知完成
      this.server.to(`session:${sessionId}`).emit('agent:complete', {
        agentId: agent.id,
        agentName: agent.name,
        fullContent,
        tokenUsage: {
          /* 从响应中获取 */
        },
        timestamp: new Date(),
      });
    } catch (error) {
      this.server.to(`session:${sessionId}`).emit('agent:error', {
        agentId: agent.id,
        agentName: agent.name,
        error: error.message,
        timestamp: new Date(),
      });
    }
  }

  /**
   * 构建带角色定位的提示词
   */
  private buildPromptWithRole(agent: ILLMAdapter, userMessage: string, context: AgentContext): string {
    return `你是 ${agent.name}，${agent.role}。

当前聊天室上下文：
${this.formatSharedMemory(context.sharedMemory)}

最近对话：
${this.formatRecentMessages(context.conversationHistory)}

用户消息：${userMessage}

请根据你的职责（${agent.capabilities.join('、')}）给出专业的回应。`;
  }
}
```

**工作空间服务**：

```typescript
@Injectable()
export class WorkspaceService {
  constructor(
    @InjectRedis() private readonly redis: Redis,
    @InjectRepository(Workspace) private readonly workspaceRepo: Repository<Workspace>,
  ) {}

  /**
   * 获取共享工作空间
   */
  async getWorkspace(sessionId: string): Promise<Workspace> {
    // 先从Redis缓存获取
    const cached = await this.redis.get(`workspace:${sessionId}`);
    if (cached) {
      return JSON.parse(cached);
    }

    // 从数据库加载
    const workspace = await this.workspaceRepo.findOne({
      where: { sessionId },
    });

    if (!workspace) {
      return this.createWorkspace(sessionId);
    }

    // 缓存到Redis
    await this.redis.setex(`workspace:${sessionId}`, 3600, JSON.stringify(workspace));

    return workspace;
  }

  /**
   * 添加文件到工作空间
   */
  async addFile(
    sessionId: string,
    file: {
      path: string;
      content: string;
      language: string;
      author: string;
    },
  ): Promise<void> {
    const workspace = await this.getWorkspace(sessionId);

    // 检查文件是否已存在
    const existingIndex = workspace.codeRepository.files.findIndex((f) => f.path === file.path);

    if (existingIndex >= 0) {
      // 更新现有文件（版本+1）
      workspace.codeRepository.files[existingIndex] = {
        ...workspace.codeRepository.files[existingIndex],
        content: file.content,
        version: workspace.codeRepository.files[existingIndex].version + 1,
        lastModifiedBy: file.author,
        updatedAt: new Date(),
      };
    } else {
      // 添加新文件
      workspace.codeRepository.files.push({
        ...file,
        version: 1,
        lastModifiedBy: file.author,
        createdAt: new Date(),
        updatedAt: new Date(),
      });
    }

    // 更新缓存和数据库
    await this.updateWorkspace(sessionId, workspace);

    // 广播文件变更事件
    this.broadcastWorkspaceChange(sessionId, 'file:added', file);
  }

  /**
   * 添加任务到工作空间
   */
  async addTask(sessionId: string, task: Partial<Workspace['tasks'][0]>): Promise<void> {
    const workspace = await this.getWorkspace(sessionId);

    workspace.tasks.push({
      id: uuidv4(),
      ...task,
      status: 'pending',
      createdAt: new Date(),
    } as any);

    await this.updateWorkspace(sessionId, workspace);
    this.broadcastWorkspaceChange(sessionId, 'task:added', task);
  }

  /**
   * 更新工作空间
   */
  private async updateWorkspace(sessionId: string, workspace: Workspace): Promise<void> {
    // 更新Redis缓存
    await this.redis.setex(`workspace:${sessionId}`, 3600, JSON.stringify(workspace));

    // 异步更新数据库
    this.workspaceRepo.save(workspace);
  }

  /**
   * 广播工作空间变更
   */
  private broadcastWorkspaceChange(sessionId: string, event: string, data: any): void {
    this.server.to(`session:${sessionId}`).emit('workspace:change', {
      event,
      data,
      timestamp: new Date(),
    });
  }
}
```

```typescript
@Injectable()
export class ShortTermMemoryService {
  constructor(@InjectRedis() private readonly redis: Redis) {}

  /**
   * 保存短期记忆

  async save(sessionId: string, messages: Message[]): Promise<void> {
    const key = `memory:short:${sessionId}`;

    // 只保留最近20条
```

```typescript
   */
  async save(message: Message): Promise<void> {
    // 1. 保存到PostgreSQL
    await this.messageRepo.save(message);

    // 2. 向量化并保存到ChromaDB
    await this.chromaClient.collection('messages').add({
      ids: [message.id],
      documents: [message.content],
      metadatas: [
        {
          sessionId: message.sessionId,
          role: message.role,
          userId: message.userId,
          agentId: message.agentId,
          createdAt: message.createdAt.toISOString(),
        },
      ],
    });
  }

  /**
   * 语义检索相关消息
   */
  async searchSimilar(query: string, sessionId: string, limit: number = 5): Promise<Message[]> {
    // 1. 向量检索
    const results = await this.chromaClient.collection('messages').query({
      queryTexts: [query],
      nResults: limit,
      where: { sessionId },
    });

    // 2. 从数据库获取完整消息
    const messageIds = results.ids[0];
    const messages = await this.messageRepo.findByIds(messageIds);

    return messages;
  }

  /**
   * 获取会话历史（分页）
   */
  async getHistory(sessionId: string, limit: number = 50, offset: number = 0): Promise<Message[]> {
    return await this.messageRepo.find({
      where: { sessionId },
      order: { createdAt: 'DESC' },
      take: limit,
      skip: offset,
    });
  }
}
```

```typescript
@Injectable()
export class SharedMemoryService {
  constructor(@InjectRedis() private readonly redis: Redis) {}

  /**
   * 获取共享记忆（所有Agent都能访问）
   */
  async get(sessionId: string): Promise<SharedMemory> {
    const key = `memory:shared:${sessionId}`;
    const data = await this.redis.get(key);

    if (!data) {
      return this.createDefault(sessionId);
    }

    return JSON.parse(data);
  }

  /**
   * 更新共享记忆（任何Agent都能修改）
   */
  async update(
    sessionId: string,
    updater: string, // 更新者（agent id 或 user id）
    updates: Partial<SharedMemory>,
  ): Promise<void> {
    const key = `memory:shared:${sessionId}`;
    const current = await this.get(sessionId);

    const updated = {
      ...current,
      ...updates,
    };

    // 更新Redis
    await this.redis.setex(
      key,
      7200, // 2小时TTL
      JSON.stringify(updated),
    );

    // 记录更新日志
    await this.logMemoryUpdate(sessionId, updater, updates);

    // 广播变更（让其他Agent感知）
    this.broadcastMemoryChange(sessionId, updater, updates);
  }

  /**
   * 添加话题到最近讨论
   */
  async addRecentTopic(sessionId: string, topic: string, keywords: string[], participants: string[]): Promise<void> {
    const memory = await this.get(sessionId);

    memory.recentTopics.push({
      topic,
      keywords,
      timestamp: new Date(),
      participants,
    });

    // 只保留最近10个话题
    if (memory.recentTopics.length > 10) {
      memory.recentTopics = memory.recentTopics.slice(-10);
    }

    await this.update(sessionId, 'system', memory);
  }

  /**
   * 记录Agent贡献
   */
  async recordAgentContribution(
    sessionId: string,
    agentId: string,
    contribution: {
      type: 'code' | 'review' | 'suggestion' | 'design';
      description: string;
    },
  ): Promise<void> {
    const memory = await this.get(sessionId);

    if (!memory.agentStates[agentId]) {
      memory.agentStates[agentId] = {
        status: 'idle',
        currentTask: '',
        lastActivity: new Date(),
        contributionCount: 0,
      };
    }

    memory.agentStates[agentId].contributionCount++;
    memory.agentStates[agentId].lastActivity = new Date();

    await this.update(sessionId, agentId, memory);
  }

  /**
   * 广播记忆变更（让所有Agent感知）
   */
  private broadcastMemoryChange(sessionId: string, updater: string, changes: Partial<SharedMemory>): void {
    // 通过Redis Pub/Sub通知所有Agent
    this.redis.publish(
      `memory:change:${sessionId}`,
      JSON.stringify({
        updater,
        changes,
        timestamp: new Date(),
      }),
    );
  }
}
```

```typescript
@Injectable()
export class MemorySummarizationService {
  constructor(private readonly llmService: LLMService) {}

  /**
   * 生成对话摘要
   */
  async summarize(messages: Message[]): Promise<string> {
    const conversationText = messages.map((m) => `${m.role}: ${m.content}`).join('\n');

    const prompt = `请总结以下对话的核心内容和关键决策，保留重要的技术细节：\n\n${conversationText}`;

    const summary = await this.llmService.generate(prompt, {
      model: 'gpt-4',
      temperature: 0.3,
      max_tokens: 500,
    });

    return summary;
  }

  /**
   * 检查是否需要摘要
   */
  async shouldSummarize(sessionId: string): Promise<boolean> {
    const messages = await this.longTermMemory.getHistory(sessionId, 100);

    if (messages.length > 50) return true;

    // 计算总token数
    const totalTokens = messages.reduce((sum, m) => {
      return sum + this.estimateTokens(m.content);
    }, 0);

    return totalTokens > 4000;
  }
}
```

## 4. LangGraph编排模块

### 4.1 模块概述

使用LangGraph管理Agent的协作流程，支持灵活的工作流编排。**注意：此模块是可选的，在群聊模式下Agent主要通过主动决策机制参与，工作流编排用于特定的结构化任务。**

### 4.2 功能需求

#### FR-4.1 自由对话模式（默认）

**需求描述**：Agent像群聊一样自由对话，无固定流程。

**特点**：

- ✅ Agent主动感知并决策
- ✅ 无固定顺序
- ✅ 支持多Agent同时响应
- ✅ 类似Slack/Discord群组讨论

**实现**：

```typescript
@Injectable()
export class FreeChatOrchestrator {
  /**
   * 处理自由对话（无编排）
   */
  async handleFreeChat(message: Message, sessionId: string): Promise<void> {
    // 1. 获取所有Agent
    const agents = this.agentConfigService.getAllAgents();

    // 2. 获取共享上下文
    const context = await this.memoryService.getSessionContext(sessionId);

    // 3. 并行让所有Agent决策
    const responses = await Promise.allSettled(agents.map((agent) => this.tryAgentResponse(agent, message, context)));

    // 4. 记录响应结果
    responses.forEach((result, index) => {
      if (result.status === 'fulfilled' && result.value) {
        console.log(`${agents[index].name} 响应成功`);
      }
    });
  }

  private async tryAgentResponse(
    agent: ILLMAdapter,
    message: Message,
    context: AgentContext,
  ): Promise<AgentResponse | null> {
    // 决策是否响应
    const decision = await agent.shouldRespond(message, context);

    if (!decision.should) {
      return null;
    }

    // 生成响应
    return await agent.generate(message.content, context);
  }
}
```

#### FR-4.2 结构化工作流模式（可选）

**需求描述**：对于特定任务（如"完整实现一个功能"），使用LangGraph编排固定流程。

**触发条件**：

- 用户明确要求："使用工作流模式"
- 管理员配置自动触发关键词

**默认工作流（三Agent协作）**：

```
   ↓

   ↓

   ↓
```

```typescript
import { StateGraph, END } from '@langchain/langgraph';

@Injectable()
export class StructuredWorkflowOrchestrator {
  /**
   * 创建结构化协作工作流
   */
  createStructuredWorkflow(): StateGraph {
    const workflow = new StateGraph({
      channels: {
        sessionId: null,
        userRequirement: null,
        claudeDesign: null, // Claude设计方案
        claudeCode: null, // Claude生成代码
        codexReview: null, // Codex审查意见
        geminiUXSuggestion: null, // Gemini UX建议
        finalOutput: null,
        sharedWorkspace: null, // 共享工作空间引用
      },
    });

    // 节点1: Claude架构设计与编码
    workflow.addNode('claude_design_and_code', async (state) => {
      const claudeAgent = this.agentConfigService.getAgent('claude-001');

      // 获取共享记忆
      const sharedMemory = await this.memoryService.getSharedMemory(state.sessionId);

      const prompt = `
        用户需求：${state.userRequirement}
        
        请作为架构师和开发者：
        1. 设计系统架构
        2. 编写完整实现代码
        3. 添加必要注释
        
        当前技术栈：${JSON.stringify(sharedMemory.techStack)}
      `;

      const result = await claudeAgent.generate(prompt, {
        sessionId: state.sessionId,
        sharedMemory,
      });

      // 更新共享工作空间
      await this.workspaceService.addFile(state.sessionId, {
        path: 'generated/main.ts',
        content: result.content,
        language: 'typescript',
        author: claudeAgent.id,
        authorName: claudeAgent.name,
      });

      return {
        ...state,
        claudeDesign: result.metadata?.design,
        claudeCode: result.content,
      };
    });

    // 节点2: Codex代码审查
    workflow.addNode('codex_review', async (state) => {
      const codexAgent = this.agentConfigService.getAgent('codex-001');

      const prompt = `
        请审查以下由Claude生成的代码：
        
        \`\`\`typescript
        ${state.claudeCode}
        \`\`\`
        
        请检查：
        1. 代码质量和规范
        2. 潜在bug和安全问题
        3. 性能优化建议
        4. 测试覆盖建议
      `;

      const review = await codexAgent.generate(prompt, {
        sessionId: state.sessionId,
      });

      return {
        ...state,
        codexReview: review.content,
      };
    });

    // 节点3: Gemini UI/UX建议
    workflow.addNode('gemini_ux_suggestion', async (state) => {
      const geminiAgent = this.agentConfigService.getAgent('gemini-001');

      const prompt = `
        基于以下实现，提供UI/UX优化建议：
        
        用户需求：${state.userRequirement}
        代码实现：${state.claudeCode}
        
        请提供：
        1. 用户体验优化建议
        2. 界面设计方案
        3. 交互流程改进
      `;

      const suggestion = await geminiAgent.generate(prompt, {
        sessionId: state.sessionId,
      });

      return {
        ...state,
        geminiUXSuggestion: suggestion.content,
        finalOutput: this.formatOutput(state, suggestion.content),
      };
    });

    // 定义流程
    workflow.setEntryPoint('claude_design_and_code');
    workflow.addEdge('claude_design_and_code', 'codex_review');
    workflow.addEdge('codex_review', 'gemini_ux_suggestion');
    workflow.addEdge('gemini_ux_suggestion', END);

    return workflow;
  }

  /**
   * 执行工作流
   */
  async executeWorkflow(sessionId: string, userRequirement: string): Promise<WorkflowResult> {
    const workflow = this.createStructuredWorkflow();
    const compiled = workflow.compile();

    // 执行
    const result = await compiled.invoke({
      sessionId,
      userRequirement,
      sharedWorkspace: await this.workspaceService.getWorkspace(sessionId),
    });

    return result;
  }
}
```

```typescript
async handleMessage(client: Socket, data: any) {
```

```typescript
@Injectable()
export class QueueService {
  private agentQueue: Queue;

  constructor() {
    this.agentQueue = new Queue('agent-tasks', {
      connection: {
        host: 'localhost',
        port: 6379,
      },
    });
  }

  /**
   * 添加Agent任务到队列
   */
  async addTask(task: AgentTask): Promise<Job> {
    return await this.agentQueue.add(
      task.type, // 任务类型
      task.data, // 任务数据
      {
        priority: task.priority || 1, // 优先级（1-10）
        attempts: 3, // 重试次数
        backoff: {
          type: 'exponential',
          delay: 2000,
        },
        removeOnComplete: false,
        removeOnFail: false,
      },
    );
  }

  /**
   * 批量添加任务
   */
  async addBulkTasks(tasks: AgentTask[]): Promise<Job[]> {
    return await this.agentQueue.addBulk(
      tasks.map((task) => ({
        name: task.type,
        data: task.data,
        opts: { priority: task.priority },
      })),
    );
  }
}
```

```typescript
@Processor('agent-tasks')
export class AgentTaskProcessor {
  constructor(
    private readonly agentService: AgentService,
    private readonly chatGateway: ChatGateway,
  ) {}

  /**
   * 处理代码生成任务
   */
  @Process('code-generation')
  async handleCodeGeneration(job: Job<CodeGenerationTask>) {
    const { sessionId, prompt, agentType } = job.data;

    // 更新任务状态
    await job.updateProgress(10);

    try {
      // 获取Agent
      const agent = this.agentService.getAgent(agentType);

      // 通知开始
      this.chatGateway.emitToSession(sessionId, 'agent:start', {
        jobId: job.id,
        agentId: agent.id,
      });

      await job.updateProgress(30);

      // 执行生成
      const result = await agent.generate(prompt, {
        sessionId,
        onProgress: async (progress) => {
          await job.updateProgress(30 + progress * 0.6); // 30%-90%
        },
      });

      await job.updateProgress(100);

      // 通知完成
      this.chatGateway.emitToSession(sessionId, 'agent:complete', result);

      return result;
    } catch (error) {
      // 通知错误
      this.chatGateway.emitToSession(sessionId, 'agent:error', {
        error: error.message,
      });

      throw error;
    }
  }

  /**
   * 处理代码审查任务
   */

  async handleCodeReview(job: Job<CodeReviewTask>) {
    // 类似实现...
  }

  /**
   * 任务完成钩子
   */
  @OnQueueCompleted()
  async onCompleted(job: Job, result: any) {
```

```typescript
@Injectable()
export class AuthService {
  async register(dto: RegisterDto): Promise<User> {
    // 1. 验证用户名/邮箱唯一性
    const existingUser = await this.userRepo.findOne({
      where: [{ username: dto.username }, { email: dto.email }],
    });

    if (existingUser) {
      throw new ConflictException('用户名或邮箱已存在');
    }

    // 2. 加密密码
    const hashedPassword = await bcrypt.hash(dto.password, 10);

    // 3. 创建用户
    const user = this.userRepo.create({
      username: dto.username,
      email: dto.email,
      password: hashedPassword,
    });

    return await this.userRepo.save(user);
  }
}
```

#### FR-7.2 用户登录

```typescript
async login(dto: LoginDto): Promise<{ accessToken: string; refreshToken: string }> {
  // 1. 验证用户
  const user = await this.userRepo.findOne({ where: { username: dto.username } });

  if (!user || !(await bcrypt.compare(dto.password, user.password))) {
    throw new UnauthorizedException('用户名或密码错误');
  }

  // 2. 生成JWT
  const payload = { sub: user.id, username: user.username, role: user.role };

  const accessToken = this.jwtService.sign(payload, { expiresIn: '15m' });
  const refreshToken = this.jwtService.sign(payload, { expiresIn: '7d' });

  // 3. 更新最后登录时间
  user.lastLoginAt = new Date();
  await this.userRepo.save(user);

  return { accessToken, refreshToken };
}
```

```typescript
@Injectable()
export class RolesGuard implements CanActivate {
  constructor(private reflector: Reflector) {}

  canActivate(context: ExecutionContext): boolean {
    const requiredRoles = this.reflector.get<string[]>('roles', context.getHandler());

    if (!requiredRoles) {
      return true;
    }

    const request = context.switchToHttp().getRequest();
    const user = request.user;

    return requiredRoles.includes(user.role);
  }
}
```

```typescript
@Injectable()
export class FileStorageService {
  private minioClient: Minio.Client;

  constructor(configService: ConfigService) {
    this.minioClient = new Minio.Client({
      endPoint: configService.get('MINIO_ENDPOINT'),
      port: configService.get('MINIO_PORT'),
      useSSL: false,
      accessKey: configService.get('MINIO_ACCESS_KEY'),
      secretKey: configService.get('MINIO_SECRET_KEY'),
    });
  }

  async uploadFile(
    bucketName: string,
    fileName: string,
    fileBuffer: Buffer,
    metadata?: Record<string, any>,
  ): Promise<string> {
    // 确保bucket存在
    const bucketExists = await this.minioClient.bucketExists(bucketName);
    if (!bucketExists) {
      await this.minioClient.makeBucket(bucketName, 'us-east-1');
    }

    // 上传文件
    await this.minioClient.putObject(bucketName, fileName, fileBuffer, fileBuffer.length, metadata);

    // 返回文件URL
    return `${bucketName}/${fileName}`;
  }
}
```

## 业务流程设计

### 典型业务流程1：自由群聊式协作（默认模式）

```
1. [用户] 通过WebSocket发送消息
   "我想实现一个用户登录功能"
   ↓
2. [聊天室模块] 接收消息，解析@提及，保存到数据库
   ↓
3. [聊天室模块] 广播消息给所有在线成员（前端用户 + 所有Agent）
   ↓
4. [决策引擎] 并行询问所有Agent：你要响应这条消息吗？
   ↓
   ├─ [Claude] shouldRespond() → Yes（涉及架构和编码）
   ├─ [Codex] shouldRespond() → No（暂时无代码需要审查）
   └─ [Gemini] shouldRespond() → Yes（可以提供UI设计建议）
   ↓
5. [前端显示]
   "Claude 正在思考...（原因：消息涉及我的专业领域）"
   "Gemini 正在思考...（原因：我可以提供UI设计建议）"
   ↓
6. [Claude Agent] 开始生成响应
   - 流式推送："我来设计登录系统的架构..."
   - 生成代码块
   - 保存到共享工作空间
   ↓
7. [Gemini Agent] 同时开始生成
   - 流式推送："从用户体验角度，登录页面应该..."
   - 提供设计建议
   ↓
8. [共享记忆更新] Claude和Gemini的响应都保存到：
   - 短期记忆（Redis）
   - 长期记忆（PostgreSQL + ChromaDB）
   - 共享工作空间（代码文件）
   ↓
9. [Codex感知变化] 检测到共享工作空间有新代码
```
